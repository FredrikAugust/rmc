# `rmc` - Ruby Music Client

## Crash course
`j` navigate up

`k` navigate down

`<space>` or `p` pause/unpause

`<enter>` play

`-` volume down by 5%

`+` volume up by 5%

`>` next song

`<` previous song

## How to run
`cd repo/lib/`

`ruby tui.rb`

## Deps.
`mopidy` this has to run on `localhost` port `6600` unless you wanna change that manually. You can do that in the `back_end.rb` file

`ruby >=2.0`

`ncurses-ruby`

`ruby-mpd`
